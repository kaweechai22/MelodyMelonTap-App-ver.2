# MelodyMelonTap 🍉🎤

แอปวิเคราะห์ความสุกของแตงโมด้วยเสียงเคาะ โดยใช้ FFT และค่า Q-factor

## ผลลัพธ์ที่แสดง
- ระดับความสุก: ดิบ / ใกล้สุก / สุกพอดี / สุกเกินหรืออาจเน่า
- ความหวาน (%Brix โดยประมาณ)
- ความกรอบ
- ความฉ่ำ
- โพรงภายใน (แสดงแยกจากความสุก)
- ความเชื่อมั่น (%)

## วิธีใช้
1. กดเริ่มฟังเสียง
2. เคาะแตงโม 3 ครั้ง
3. รอผลลัพธ์
4. แชร์ผลหรือบันทึกไว้ได้

## หมายเหตุ
- ควรใช้ในที่เงียบ
- ค่าที่ได้เป็นค่าประมาณจากโมเดล
